♥ Paragraphs ♥

A paragraph consists of one or more sentences that form a self contained unit of discourse. Everytime we need the text to start on the next line (a new paragraph) we should write the tags: ‹p› and ‹/p› and we are setting to start displaying the text in the following line.

♥ Images ♥

This is getting boring, right? Too much text! Let's add some imagessss =D

To add an image into the page you need to use an ‹img› element. This is an empty element (which means there is no closing tag). It must carry the following two attributes:

  • src (this tells the browser where it can find the image file [url])
  • alt (this provides a text description of the image which describes the image file if you cannot see it)

♥ Tables ♥

The ‹table› element is used to create a table and the contents of the table are written out row by row.

You indicate the start of each row using the opening tag ‹tr›. This should be followed by one or more ‹td› elements (one for each cell in that row) But what is the ‹td› element?!

Each cell of a table is represented using a ‹td› element (The td stands for table data.)
